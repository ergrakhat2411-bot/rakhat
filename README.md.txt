# 🚀 Telegram Bot - Портфолио Ергазина Рахата

Интерактивный Telegram-бот, разработанный на асинхронной библиотеке **aiogram 3.x**. Бот заменяет привычное консольное портфолио, позволяя пользователям удобно просматривать информацию обо мне, моих целях, хобби и выполненных проектах прямо в мессенджере.

---

## 🛠️ Технологический стек

* **Язык программирования:** Python 3.9+
* **Фреймворк:** [aiogram 3.x](https://github.com/aiogram/aiogram) (Асинхронная библиотека для работы с Telegram Bot API)
* **Хранение данных:** JSON (внешний файл данных `data_2.json`)
* **Логирование:** Модуль `logging`

---

## 📦 Архитектура проекта

Проект спроектирован с разделением логики и контента (без хардкода ссылок и текстов в самом коде):
* `main_2.py` — основной файл бота, содержащий асинхронные обработчики (хендлеры) и клавиатуры.
* `data_2.json` — структурированная база данных в формате JSON, хранящая всю текстовую информацию, списки проектов и ссылки.

---

## 🌟 Основные возможности и функции

1. **Главное меню (Reply Keyboard):** Быстрый доступ к 8 ключевым разделам портфолио в один клик.
2. **Разделы «О себе» и «Цели»:** Форматированный вывод личной информации с использованием HTML-тегов (`bold`, `italic`) и эмодзи.
3. **Интерактивный вывод проектов:** Бот динамически считывает список проектов из JSON-файла и генерирует персональные `Inline-кнопки` со ссылками на работы.
4. **Кнопка-ссылка на GitHub:** Быстрый переход в репозиторий автора через инлайн-клавиатуру.
